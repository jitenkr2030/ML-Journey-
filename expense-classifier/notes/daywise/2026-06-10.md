## Machine Learning Concepts Learned

### 1. Text Data Cannot Be Used Directly

Computers cannot understand text.

Example:

Salary Transfer

is a string, not a number.

Machine learning models require numerical input.

Therefore text must be converted into numbers before training.

---

### 2. TF-IDF Vectorization

Used:

```python
TfidfVectorizer()
```

Purpose:

Convert text into numerical vectors.

Example:

Uber Ride

becomes:

[0.12, 0.53, 0.21, ...]

The model learns from these numerical representations.

Key Learning:

TF-IDF acts as a translator between human language and machine learning algorithms.

---

### 3. Naive Bayes Algorithm

Used:

```python
MultinomialNB()
```

Purpose:

Classify text into categories based on probability.

Example:

Salary Transfer

contains the word:

Salary

which frequently appears in Payroll records.

Therefore the model predicts:

Payroll
