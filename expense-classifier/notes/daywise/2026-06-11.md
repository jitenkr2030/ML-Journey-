## Machine Learning Concepts Learned

### 1. Why More Data Matters

Machine learning learns patterns from examples.

Small dataset:

* Poor predictions
* Low accuracy

Large dataset:

* Better learning
* Better predictions
* Better accuracy

Important lesson:

More quality data usually improves machine learning performance.

### 3. TF-IDF Vectorization

Learned how text is converted into numbers.

Example:

Input:

Uber Airport Ride

Machine learning cannot understand text directly.

TF-IDF converts words into numerical vectors.

Workflow:

Text
→ TF-IDF
→ Numerical Features
→ Model Training
