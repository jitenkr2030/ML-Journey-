# What We Learned

## 1. Model Deployment Basics

Machine Learning is not only about training models.

A trained model becomes useful when users can interact with it.

Today we learned:

* Load a saved model
* Load a saved TF-IDF vectorizer
* Accept user input
* Convert text into numerical features
* Predict expense categories
* Display results

---

## 2. Loading Saved Models

We learned how to load saved files using Joblib.

```python
import joblib

model = joblib.load(
    "expense_classifier_day7.pkl"
)

vectorizer = joblib.load(
    "tfidf_vectorizer_day7.pkl"
)
```

Purpose:

* Avoid retraining every time
* Use the trained model instantly
* Faster predictions

---

## 3. Taking User Input

We learned how to collect input from users.

Example:

```python
expense = input(
    "Enter Expense Description: "
)
```

This allows users to enter:

* Travel expenses
* Payroll expenses
* GST expenses
* Software expenses

and get predictions immediately.

---

## 4. Converting Input to TF-IDF Features

The model cannot understand text directly.

We learned how to transform user text using the saved vectorizer.

Example:

```python
expense_vector = vectorizer.transform(
    [expense]
)
```

This converts text into numbers.

---

## 5. Making Predictions

We learned how to generate predictions.

Example:

```python
prediction = model.predict(
    expense_vector
)
```

Output:

```python
Travel
Software
Payroll
GST
```

---

## 6. Infinite Loop Applications

We learned how to keep the application running.

Example:

```python
while True:
```

Benefits:

* Multiple predictions
* Better user experience
* No need to restart application

---

## 7. Exit Commands

We learned how to terminate the application safely.

Example:

```python
if expense.lower() == "exit":
    break
```

This creates a professional CLI application.
